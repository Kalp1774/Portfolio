# Hacker Portfolio

A personal portfolio website with a unique **Hacker / Linux Desktop** aesthetic. This project simulates an interactive operating system in the browser, complete with a boot sequence, terminal, file explorer, and functional applications.

##  Features

*   **Immersive Boot Sequence**: "Hacker-style" boot log animation with a "YOU ARE BEING WATCHED" security alert.
*   **Interactive Desktop Environment**:
    *   Draggable windows.
    *   Taskbar with start menu and running apps.
    *   Desktop icons with white text labels.
    *   Custom "RAW/CBI" intelligence agency themed wallpaper.
*   **Functional Applications**:
    *   **Terminal**: Execute commands like `help`, `ls`, `cat`, `whoami`, `date`, `clear`, and `hack`.
    *   **File Explorer**: Navigate directories and view files (About, Projects, Contact).
    *   **About Me**: Displays user profile with a "Classified Agent" photo style.
    *   **Snake Game**: Classic Snake game with score tracking.
    *   **Paint**: A drawing application with color picker and brush size control.
    *   **Calculator**: Fully functional standard calculator.
    *   **Tic-Tac-Toe**: Playable Tic-Tac-Toe game against a local opponent.
*   **Visual Effects**:
    *   Matrix rain background effect.
    *   CRT monitor scanline styling.
    *   Glow effects and monospace fonts (Ubuntu, Fira Code).

##  Technologies Used

*   **HTML5**: Structure and semantic markup.
*   **CSS3**: Styling, animations, grid/flexbox layouts, and variables.
*   **JavaScript (ES6+)**: Core logic for the OS simulation, window management, and application functionality.
*   **Font Awesome**: Icons for the desktop and UI.
*   **Google Fonts**: Ubuntu and Fira Code fonts.

##  How to Run

1.  Clone or download this repository.
2.  Open `index.html` in any modern web browser.
3.  Enjoy the experience!

##  Customization

*   **Wallpaper**: Replace `assets/wallpaper.png` with your own image.
*   **Profile Photo**: Replace `assets/profile.jpg` with your own photo.
*   **Content**: Edit `main.js` to modify the file system content (About, Projects, Contact info).
